What is YQL?

YQL stands for Yahoo! Query Language, which is a query language that was created by Yahoo! for developmental purposes. If you are familiar with SQL queries, then you will quickly catch on as YQL has very similar structures to SQL. YQL retrieves and manipulates data from a variety of different API's for the convenience of the developer into one single web interface. When you run a query on YQL, you will receive a weblink with a GET request with response information in either XML or JSON format, depending on what you choose. With the ability to access an enormous amount of information with one source, developers can focus on developing their software and worry less on the back-end data.

Who Should Read This Guide

This guide is aimed towards developers interested in looking for a general purpose information gathering API powered by Yahoo! for responses in their back-end with ease by using GET requests and parsing JSON or XML as responses. Developers can use YQL to create a wide array of applications ranging from a simple weather app, to a finance app by retrieving stock market information.

Purpose Of This Guide

This guide will be a quick starter-guide to help developers dive right into YQL without too much overhead. Readers will quickly learn the basic methods and queries to retrieve information from YQL. This guide is not meant to be go over every single facet and detail of the YQL API, but rather a quick starting point with enough information for readers to explore YQL. This guide will start with introduction of the YQL developer console layout, followed by a few examples to show users how to to quickly begin making YQL queries.

Prerequisites

  • Understanding of how to do a GET response in HTTP
  • Understanding of how to parse JSON or XML data
  • Understanding of how to build URL strings in user's program
  • Basic understanding of SQL queries
